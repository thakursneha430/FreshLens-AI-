## 🚀 Features
🌟 Upload food images easily<br>
🧠 Deep Learning CNN model for spoilage detection<br>
🌐 Clean and simple web interface<br>
📁 Well-organized project structure

## 🧠 How It Works
1️⃣ User uploads a food image<br>
2️⃣ Image gets preprocessed<br>
3️⃣ CNN model predicts freshness<br>
5️⃣ Result displayed on web app<br>
